# Pomodoro Timer

> A clean, fully-featured **focus timer** built with pure HTML, CSS, and JavaScript — zero dependencies.

---

## Features

- **3 modes** — Focus (25m), Short Break (5m), Long Break (15m)
- Animated SVG ring progress indicator
- Color-coded themes per mode
- Auto-advance — transitions automatically between focus and break
- Session dot tracker — 4 dots light up before a long break
- Adjustable durations via on-screen steppers
- Web Audio API beep alert on timer completion
- Browser notification support
- Session history log with timestamps
- **No install needed** — open `index.html` directly

---

## Run

Just open `index.html` in any browser. No build step, no npm, no server.

```bash
git clone https://github.com/Manzi-ol/pomodoro-timer
# Open index.html in your browser
```

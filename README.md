# Pomodoro Timer

Project #13 of Manzi's 100 GitHub Projects Roadmap.

A clean, fully-featured Pomodoro focus timer built with vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies.

## Features

- Focus / Short Break / Long Break modes with color-coded themes
- Animated SVG ring progress indicator
- Auto-advance: transitions from focus → break → focus automatically
- Session counter with dot progress (every 4 sessions = long break)
- Adjustable durations via steppers (1-minute increments)
- Web Audio API beep alert on completion
- Browser notification support
- Session history log with timestamps
- Zero dependencies — open `index.html` directly in any browser

## Usage

Just open `index.html` in your browser. No build step required.

```
pomodoro-timer/
├── index.html
├── style.css
└── app.js
```

## Default Durations

| Mode | Default |
|------|---------|
| Focus | 25 min |
| Short Break | 5 min |
| Long Break | 15 min |
